You can use Riemann sums to estimate the net area, where you approximate the enclosed region with rectangles using left or right endpoints or midpoints to determine the height,˙ or you can select Simpson’s Rule or the Trapezoidal Rule. In each case, the module will compute the estimated value and display a graph of the area computed. You can improve the approximation by increasing the number of subintervals, and you can see the error in the estimated value both graphically and numerically.
